## Installation

1. Clone
2. Install Depenndencies

```bash
npm install
```

3. Create database

```bash
sequelize db:create
```

4. Migrate database

```bash
sequelize db:migrate
```

5. Jalankan seeder database

```bash
sequelize db:seed:all
```

6. Jalankan aplikasi

```bash
npm test
```
